RxCheck : Viewing, Deleting, and Searching an Existing Datasource

Created by Joseph Fibich on Jan 13, 2026

  1. Follow steps 1, 2, and 4 under the section titled View and add a datasource for provider validation in this guide to view an existing datasource.
    The new screen will look similar to the Add Datasource screen, but includes a section labeled Search Provider.

  1. To see if an existing datasource file contains a provider, you can enter the DEA#, NPI#, SL# or OTHER# in the Search Provider section to see if that provider exists on the datasource you are viewing.

    1. If the provider is on the list, you will see their information returned under the Search Provider section.

    2. If the provider is not on the list, you will see “No Provider found.” Under the Search Provider section.

The datasource search feature is only available for datasource files and cannot be used on a datasource API.
