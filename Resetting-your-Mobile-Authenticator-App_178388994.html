RxCheck : Resetting your Mobile Authenticator App

1. Users can set up or reset their authenticator app by:

a. Scanning the QR code from their RxConsole account creation email, or

b. Clicking Reset QR Code on the authenticator code screen

2. If you clicked the Reset QR Code link, press the Email QR Code button on the following screen.

3. A Reset QR Code – Email Sent pop-up will display. Press the Ok button.

4. Users will receive an email from “prod-notif” with the subject Reset QR Code .

5. Open the email and scan the QR Code.

The QR code here is for reference only.

If the user previously set up authentication in a mobile authenticator app, they should delete the existing RxConsole account entry in that authenticator app before scanning the new QR code to avoid duplicate entries.
